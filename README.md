# SudokuSolver
# C++ program for 9*9 Sudoku
## Download the file and execute it.
#### Directly feed the SUDOKU MATRIX
### Format of Sudoku Matrix:-
-> "__"(blank spaces) to be feeded as "0"
-> Already filled no. as it is .
# eg:
5 3 0 0 7 0 0 0 0     
6 0 0 1 9 5 0 0 0     
0 9 8 0 0 0 0 6 0     
8 0 0 0 6 0 0 0 3     
4 0 0 8 0 3 0 0 1      
7 0 0 0 2 0 0 0 6      
0 6 0 0 0 0 2 8 0      
0 0 0 4 1 9 0 0 5      
0 0 0 0 8 0 0 7 9     
## Horray! Sudoku will be solved.
